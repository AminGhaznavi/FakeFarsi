# پکیج FakeFarsi (نسخه 0.0.8+)

پکیج **FakeFarsi** یک بسته Python است که به شما امکان می‌دهد اطلاعات جعلی و متنوعی - شامل نام، نام خانوادگی، سن، شماره تلفن، ایمیل و سایر جزئیات فرد خیالی ایرانی - تولید کنید. این پروژه به منظور ساده‌سازی فرآیند تولید پروفایل‌های ساختگی و معتبرسازی آنها بومی‌سازی شده برای کاربران و توسعه‌دهندگان ایرانی توسعه یافته است.

---

## ✨ ویژگی‌ها (بهینه‌سازی شده)

* **تولید نام و نام خانوادگی فارسی:** تولید نام‌های تفکیک‌شده بر اساس جنسیت، با نگارش فارسی و انتقال هوشمند آنها به حروف انگلیسی (فینگلیش).
* **انتخاب سن تصادفی:** تولید سن در بازه ۱۰ تا ۹۹ سال.
* **ساخت شماره تلفن‌های معتبر:** ساخت شماره‌های موبایل اپراتورهای مختلف ایرانی با پیش‌شماره‌های رایج (مثل ۰۹۱۲، ۰۹۳۵ و...).
* **تولید ایمیل‌های هوشمند:** ساخت ایمیل بر اساس نام و فامیل تولید شده با دامنه‌های رایج.

---

## 🛠️ تغییرات و قابلیت‌های جدید (توسعه یافته توسط Amin Ghaznavi)

در به‌روزرسانی اخیر، ساختار پروژه برای تجربه کاربری بهتر توسعه‌دهندگان ارتقا یافته است:
1. **هوشمندسازی متد ایمیل (`fake_email`):** این متد اکنون پارامترهای اختیاری می‌پذیرد. اگر نام و فامیل به آن پاس داده نشود، متد به صورت خودکار خودش اقدام به تولید نام و فامیل کرده و ایمیل منطبق با آن را با کمک تابع مترجم (`translator`) می‌سازد.
2. **اضافه شدن ابزار خط فرمان (CLI Menu):** یک منوی تعاملی کامل به پروژه اضافه شده تا کاربران بتوانند بدون نیاز به نوشتن کدهای پایتون، مستقیماً از طریق ترمینال داده‌های فیک را به صورت تکی یا در قالب پروفایل کامل دریافت کنند.
3. **پشتیبانی از دستور سیستمی:** با تنظیم فرآیند اِنتری‌پوینت در فایل `pyproject.toml`، پکیج اکنون قابلیت اجرا شدن با دستور مستقیم `fakefarsi` در ترمینال را داراست.

## 📦 نصب و راه‌اندازی

در حال حاضر، FakeFarsi به صورت یک بسته نصب‌پذیر در PyPI قرار دارد. برای نصب، از دستور زیر استفاده کنید:

```bash
pip install fakefarsi
```
💻 نحوه استفاده
۱. استفاده به عنوان کتابخانه در کد پایتون
from fakefarsi import FakeFarsi

ff = FakeFarsi()

print(ff.fake_name())        # نام کوچک جعلی
print(ff.fake_last_name())   # نام خانوادگی جعلی
print(ff.fake_age())         # سن جعلی
print(ff.fake_phone())       # شماره تلفن جعلی

# ساخت ایمیل به صورت هوشمند و بدون نیاز به پاس دادن اجباری ورودی
print(ff.fake_email())       

# تولید پروفایل کامل متصل به هم
print(ff.generate_fake_profile())
۲. استفاده به عنوان ابزار ترمینال (CLI)
پس از نصب، کافی است در خط فرمان خود دستور زیر را تایپ کنید تا منوی تعاملی برنامه برای شما باز شود:
fakefarsi
